libcards
========

Base on Linux system, a class library using C++ encapsulation.

It provides the following functionalities:
Complete communication mechanism between cards (e.g. data synchronization, notification).
Basic administration of the card (e.g. card activation/ shut-down, SW download, upload of diagnostic data).
Generic functions like logging, alarming or task supervision.
Basic operating system encapsulating interface (e.g. tasks, semaphores, message queues).
Basic hardware driver encapsulating interface.
